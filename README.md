# Prompt Lab

**E-book com 50 prompts de IA prontos para usar** — do zero ao resultado, sem complicação.

## Conteúdo do repositório

| Arquivo | Descrição |
|---|---|
| `Prompt_Lab_Ebook_v4.pdf` | E-book final em PDF (31 páginas) |
| `index.html` | Landing page para venda (HTML único, sem dependências) |
| `gerar_ebook.py` | Script Python que gera o PDF via ReportLab |

## Categorias do e-book

1. 💼 **Trabalho e Produtividade** — 8 prompts
2. 📚 **Estudos e Aprendizado** — 8 prompts
3. 💰 **Finanças Pessoais** — 8 prompts
4. 🏠 **Dia a Dia e Organização** — 9 prompts
5. 📱 **Redes Sociais e Conteúdo** — 9 prompts
6. 🚀 **Negócios e Empreendedorismo** — 8 prompts

**+ Bônus:** Como criar seus próprios prompts (método dos 5 elementos)

## Como gerar o PDF

```bash
# Instalar dependências
pip install reportlab pdfplumber

# Gerar o ebook
python gerar_ebook.py
```

## Landing Page

Abra `index.html` direto no browser. Para publicar, basta hospedar o arquivo em qualquer serviço estático (Netlify, GitHub Pages, Vercel).

**Antes de publicar:** substitua `SEU_LINK_KIWIFY_AQUI` no `index.html` pelo link real do produto na Kiwify.
